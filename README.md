# BBNG
BBong Web Game

## 설치
```bash
$ git clone https://github.com/westofsky/BBNG.git
$ cd BBNG
$ npm install
```

### FrontEnd
```bash
$ cd frontend/
$ npm install
```

### BackEnd
```bash
$ cd backend/
$ npm install
```
- backend/ 에 .env 파일 생성하여 DB_URL = '자신의 mongodb cluster url'을 추가하여야 합니다.

### Environment
```bash
$ (root) npm start
```
